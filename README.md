# 🏏 Dart Cricket Game

A fun dart-based cricket game where you spin a wheel and score runs!

## Prerequisites
- **Node.js** installed on your machine
- **npm** package manager

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Server
```bash
node server.js
```

The server will run on `http://localhost:3000`

## How to Play

1. Open the game in your browser
2. Choose between **Single Player** or **Multiplayer** mode
3. Click the wheel to spin it
4. Score runs based on where the pointer lands:
   - **1, 2, 3, 4, 6** - Score those runs
   - **OUT** - Lose a wicket
5. Get 5 wickets and you're ALL OUT!
